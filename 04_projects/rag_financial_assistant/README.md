# 💰 AI Financial Stress Assistant (RAG-based NLP System)

A modular NLP system that helps users with financial stress guidance using hybrid retrieval techniques.

## 🚀 Features
- Semantic search using embeddings
- Keyword-based retrieval using TF-IDF
- Hybrid ranking system
- Context-aware responses
- Streamlit web interface

## 🧠 Tech Stack
- Python
- Sentence Transformers
- Scikit-learn (TF-IDF)
- Streamlit

## 📌 Key Learning
Instead of jumping directly to LLMs, this project focuses on understanding how real NLP systems work internally by combining multiple techniques.

## ⚙️ Tech Stack
- Sentence Transformers
- TF-IDF
- Streamlit

## 🧠 How It Works
1. User query → embedding
2. Retrieve relevant knowledge
3. Rank results
4. Generate structured response

## 🚀 Features
- Hybrid retrieval system
- Context-aware responses
- Simple chat interface

## 📌 Example
Input:
"I am stressed about money"

Output:
- Track expenses
- Avoid unnecessary loans

## ⚙️ System Workflow

User Query  
↓  
Embedding Generation  
↓  
Hybrid Retrieval (TF-IDF + Semantic)  
↓  
Ranking & Filtering  
↓  
Response Generation

## ⚠️ Limitations

- No true reasoning (no LLM)
- Limited knowledge base
- Basic conversational memory

## 🚀 Future Improvements

- Add intent detection
- Improve ranking logic
